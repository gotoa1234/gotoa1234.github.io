<h1>第一部分：問題描述</h1>



<h2>Step 1：NetFlix 非同住戶登入時的狀況</h2>
在 **標準、高級** 方案中，常常會有多人共用一個帳號的情況，但 **非主用戶(刷信用卡的主帳號)** 的其他人使用 Netflix 時，會出現以下訊息：
<br/> <img src="/assets/image/Infomation/2026_08_01/000.png" alt="" width="50%" height="50%" />
<br/>
<br/>這時 Netflix 主帳號的註冊信箱就會收到以下信件 
<br/> <img src="/assets/image/Infomation/2026_08_01/000_4.png" alt="" width="50%" height="50%" />
<br/><br/> 


<h2>Step 2：Netflix 原理推論</h2>
Netflix 的「同戶裝置」針對「電視（TV）或電視遊樂器等固定裝置」(不包含行動裝置)。
<br/>這時系統會偵測這台電視連線的網路 IP 地址、裝置 ID 與帳號活動。

``` Markdown
因此共享帳號的朋友們在另一個住處用他的電視登入你的帳號，
Netflix 就會偵測到兩個不同的家庭網絡（IP 區塊），進而鎖定並要求「確認同戶裝置」。
```

<br/>
<br/>Netflix <a href="https://help.netflix.com/zh-tw/node/127200">官方網站</a>，有針對 **Netflix 顯示「您的電視/裝置尚未設為同戶裝置」** 進行解決步驟，並未說明原理。
<br/> <img src="/assets/image/Infomation/2026_08_01/000_3.png" alt="" width="50%" height="50%" />
<br/><br/> 

<h2>Step 2：NetFlix 方案差異</h2>
Netflix <a href="https://help.netflix.com/zh-tw/node/24926">官方收費說明</a>，2026/7/13 日至今的方案如下，因此這篇解決方案是針對 **標準、高級** 方案的用戶， **基本** 方案只能個人使用不會有效果

<h1>第二部分：主用戶 Google Gmail 設定</h1>

<h2>Step 1：主用戶信箱操作 - 進入設定</h2>
登入主用戶的 Google Gmail (綁定 Netflix 的 Email 信箱)

``` Markdown
1. 右上角點擊 - 設定圖示
2. 點擊 - 查看所有設定
```

<br/> <img src="/assets/image/Infomation/2026_08_01/001_0.png" alt="" width="50%" height="50%" />
<br/><br/>

<h2>Step 2：主用戶信箱操作 - 新增轉寄對象</h2>
為了要自動將主用戶收到的存取信，自動轉寄給非同住戶的信箱，要依照以下步驟設定：

``` Markdown
1. 點擊 - 轉寄和 POP/IMAP
2. 點擊 - 新增轉寄地址
```

<br/> <img src="/assets/image/Infomation/2026_08_01/002.png" alt="" width="50%" height="50%" />
<br/><br/>

<h2>Step 3：主用戶信箱操作 - 新增轉寄對象</h2>
同意設定

``` Markdown
1. 點擊 - 繼續
```

<br/> <img src="/assets/image/Infomation/2026_08_01/003.png" alt="" width="50%" height="50%" />
<br/><br/>

<h2>Step 4：非同住戶操作 - 收到轉寄許可</h2>
這時非同住戶的信箱會收到 **轉寄確認** 
<br/> <img src="/assets/image/Infomation/2026_08_01/005.png" alt="" width="100%" height="100%" />
<br/><br/>

<h2>Step 5：非同住戶操作 - 允許</h2>
**非同住戶** 打開此信，並授予主用戶轉寄權限，未來就開通了 
<br/> <img src="/assets/image/Infomation/2026_08_01/006.png" alt="" width="50%" height="50%" />
<br/> 
<br/> <img src="/assets/image/Infomation/2026_08_01/007.png" alt="" width="50%" height="50%" />
<br/><br/>

<h2>Step 6：非同住戶操作 - 開通成功</h2>
最終若完成會跳出確認成功，非同住戶要操作的部分就完成了
<br/> <img src="/assets/image/Infomation/2026_08_01/008.png" alt="" width="50%" height="50%" />
<br/><br/>

<h2>Step 7：主用戶信箱操作 - 篩選器設定開啟</h2>
回到 **主用戶** 操作，並非所有信件都要轉送給 **非同住戶** ，因此我們要建立篩選器將 Netflix 的信件轉發，依序選擇：

``` Markdown
1. 點擊 - 篩選器和封鎖的地址
2. 點擊 - 建立新篩選器
```

<br/> <img src="/assets/image/Infomation/2026_08_01/009.png" alt="" width="50%" height="50%" />
<br/><br/>

<h2>Step 8：主用戶信箱操作 - 篩選器設定內容</h2>
篩選器的關鍵詞可以在主旨輸入 **Netflix 暫時存取碼**
<br/> <img src="/assets/image/Infomation/2026_08_01/010_2.png" alt="" width="50%" height="50%" />
<br/><br/>


<h2>Step 9：主用戶信箱操作 - 篩選器設定轉寄</h2>
新增轉寄地址，添加非同住戶信箱 (對方已授權才會出現下拉式選項)

``` Markdown
1. 點擊 - 新增轉寄地址
2. 點擊 - 建立新篩選器
```

<br/> <img src="/assets/image/Infomation/2026_08_01/010.png" alt="" width="50%" height="50%" />
<br/><br/>

<h2>Step 10：主用戶信箱操作 - 篩選器設定完成</h2>
保存後，最後可以按下 **建立篩選器** 即建立成功
<br/> <img src="/assets/image/Infomation/2026_08_01/011.png" alt="" width="50%" height="50%" />
<br/>
<br/>後續仍可對這個篩選器進行編輯
<br/> <img src="/assets/image/Infomation/2026_08_01/013.png" alt="" width="50%" height="50%" />
<br/><br/>

<h2>Step 11：主用戶信箱操作 - 轉寄最終設定</h2>
最後回到 **轉寄和 POP/IMAP** ， **將轉寄外來副本給** 設定到 **非同住戶的信箱**，大功告成
<br/> <img src="/assets/image/Infomation/2026_08_01/014.png" alt="" width="50%" height="50%" />
<br/><br/>
