<h2>Step 2：PowerToys 介紹</h2>
<a href="https://learn.microsoft.com/zh-tw/windows/powertoys/">Microsoft PowerToys</a> 是由微軟官方與開源社群共同維護的「Windows 系統超級外掛集」

``` Markdown
定位：進階使用者（Power Users）解鎖 Windows 預設沒有、但極度實用的生產力功能(2026/7/20 v0.100.2 的版本約有 30 種)
特性：完全免費、開源、無廣告，且整合度極高
```

<br/>這裡著重說明螢幕文字複製：在任何無法複製文字的畫面上（如圖片、防複製網頁、PDF 或影片）可以複製文字。
<br/>※最終直接用滑鼠框選，文字就會自動辨識並存入剪貼簿。

<h1>第四部分：多國語系切換 & 驗證韓文</h1>

<h2>Step 1：韓國語系測試 - 無法辨識</h2>
但英文是預設語言，辨識上是沒問題的 (中文可正常辨識 OCR 是因為作業系統預設的安裝語言)
<br/>嘗試用韓國語系辨識，就會出現以下異常，無法辨識的亂碼
<br/> <img src="/assets/image/Infomation/2026_08_15/006.png" alt="" width="50%" height="50%" />
<br/><br/> 

<h2>Step 2：原因說明</h2>
開啟 PowerShell 輸入以下指令

``` PowerShell
Get-WindowsCapability -0nline | Where-0bject { $_.Name-Like 'Language.OCR*'}
```

<br/>可以觀察到 ko-KR 的 State : NotPresent ，表示未安裝
<br/> <img src="/assets/image/Infomation/2026_08_15/007.png" alt="" width="50%" height="50%" />
<br/><br/> 


<h2>Step 3：安裝語系官方說明</h2>
<a href="https://learn.microsoft.com/zh-tw/windows/powertoys/text-extractor#install-an-ocr-language-pack">微軟官方</a>有說明可以輸入安裝，但是只提供了 **預設語法 : 英文** ，並沒有說明其他國家的語言如何安裝

``` PowerShell
$Capability = Get-WindowsCapability -Online | Where-Object { $_.Name -Like 'Language.OCR*en-US*' }
```

<br/>需要使用者自行去找出其他國家國碼
<br/> <img src="/assets/image/Infomation/2026_08_15/008.png" alt="" width="50%" height="50%" />
<br/><br/> 


<h2>Step 4：查找韓國代碼</h2>
進入微軟的<a href="https://learn.microsoft.com/zh-tw/office/2016/language/language-identifiers-optionstate-id-values">語言標識碼說明頁</a>
<br/>可以找到其他國家的辨識碼，韓國為 ko-KR
<br/> <img src="/assets/image/Infomation/2026_08_15/009.png" alt="" width="100%" height="100%" />
<br/><br/> 


<h2>Step 5：PowerShell 進行安裝</h2>
再次開啟 PowerShell 並且使用 **管理員模式開啟** 
<br/>先輸入以下：

``` PowerShell
$Capability = Get-WindowsCapability -Online | Where-Object { $_.Name -Like 'Language.OCR*ko-KR*' }
```

<br/>
<br/>然後在輸入以下指令進行正式安裝：

``` PowerShell
$Capability | Add-WindowsCapability -Online
```

<br/> <img src="/assets/image/Infomation/2026_08_15/010.png" alt="" width="100%" height="100%" />
<br/><br/> 



<h2>Step 6：安裝完成</h2>
依照網路速度的差異，可能要一些時間，最後會出現以下訊息，表示安裝完成：

``` Markdown
Path：
Online：True 
RestartNeeded：Fa1se
```

<br/> <img src="/assets/image/Infomation/2026_08_15/010_2.png" alt="" width="50%" height="50%" />
<br/><br/> 


<h2>Step 7：檢查安裝 - 重新開機</h2>
輸入以下指令進行檢查

``` PowerShell
Get-WindowsCapability -Online | Where-Object { $_.Name -Like 'Language.OCR*ko-KR*' }
```

<br/>可以觀察到 ko-KR 已經變為 Installed，這時需要重新開機，使其生效
<br/> <img src="/assets/image/Infomation/2026_08_15/011.png" alt="" width="100%" height="100%" />
<br/><br/> 


<h2>Step 8：開啟 PowerToys 設定 - 切換語系</h2>
重新開機後，回到 PowerToys 的文字擷取設定頁面， **慣用語言** 的地方出現了韓文，進行選擇
<br/> <img src="/assets/image/Infomation/2026_08_15/012.png" alt="" width="100%" height="100%" />
<br/><br/> 


<h2>Step 9：韓文擷取 - 驗證成功</h2>
最後再次執行 **Win + Shift + T** 進行韓文的擷取，可以觀察到可正常複製韓文
<br/>實現了多國語系切換的 OCR 文字擷取
<br/> <img src="/assets/image/Infomation/2026_08_15/013.png" alt="" width="50%" height="50%" />
