<h1>第一部分：Trancy 軟體介紹</h1>

<h2>Step 1：什麼是 Trancy</h2>
以下是 Trancy 的軟體簡介，本篇著重介紹如何安裝到使用，功能龐大，很值得自行深入探索

``` Markdown
Trancy 是一款基於 AI 技術打造的「全方位沉浸式語言學習助手」。

它的核心理念是讓使用者「不再枯燥地死背單字」，而是透過日常觀看的 YouTube 影片、Netflix 影劇、Udemy 線上課程，或是瀏覽外文網頁時，自然而然地融入語言的輸入（聽、讀）與輸出（說、寫）練習。

不論你正在準備多益（TOEIC）、托福（TOEFL）、雅思（IELTS），還是日檢（JLPT）或韓檢（TOPIK），Trancy 都能將整個網路世界轉化為你的個人化教材。
```

<h2>Step 2：四個核心功能</h2>
以下是 Trancy 的四個主打功能：

{:class="table table-bordered"}
| 1. 影音平台 AI 雙語字幕 | Trancy 支援 YouTube、Netflix、Disney+、Udemy、Coursera 等主流平台。 |
|                        | 基本的雙語字幕對照外，最厲害的是「閱讀模式」：會將影片縮小到一側，並在右側將所有字幕像「文章」一樣排列，方便你快速瀏覽上下文、分析句型。 |
| 2. 沉浸式網頁翻譯 | 瀏覽外文網站（如 Medium、Twitter）時，Trancy 提供段落對照翻譯。遇到不會的單字，滑鼠點擊即可即時查詢。 |
|                 | 結合了 OpenAI（ChatGPT）與 DeepL 等頂級翻譯引擎，除了單字釋義，還會直接幫你做 AI 文法分析 與 詞性標註，拆解複雜的長難句。 |
| 3. 聽力與口說強化  | 可以針對特定難句進行重複播放、調整語速。 |
|                  | 內建的 「AI 跟讀機制」，對著麥克風模仿發音後，AI 會即時針對你的發音、語調進行評分並給予優化建議。 |
| 4. 個人化生字庫與隨身 AI 家教 | （Premium 功能）看影片或網頁時儲存的生字與例句，會自動同步到雲端學習中心。 |
|                  | Premium 方案更內建了 AI 助手「Mr. Trancy」與「AI Talk」，能模擬真實生活情境與你進行語音對話，讓你零壓力練習口說。 |

<h2>Step 11：Netflix 補充 - 無法啟用的狀況排除</h2>
在 Step9 , Step10 出現無法在 Netflix 啟用的狀況時，進入 Chroem 設定 :

``` markdown
chrome://settings/system
```

<br/>設定 -> 系統 -> **關閉** 使用圖形加速功能(如果可用)
<br/>避免被 Chrome 影響 Trancy 插件
